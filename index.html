Project on Console

Visual Studio Code is a free source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

How to set VS Code

Go to https://code.visualstudio.com/Download. Choose the operating system (Microsoft Windows, Linux or macOS). OS Requirements: Microsoft Windows: Windows 7 and later, macOS: OS X 10.10 Yosemite or later.
